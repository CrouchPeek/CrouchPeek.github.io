## 🔒 Privacy Policy (Политика конфиденциальности)

1. Бот может логировать сообщения на серверах **(только с согласия владельца сервера)**.
2. Личные сообщения (ЛС) **НЕ логируются**.
3. Логируются: текст сообщений, автор, канал и время отправки.
4. Данные хранятся в зашифрованном виде и **не передаются третьим лицам**.
5. Владелец сервера может отключить логирование контента или всё логирование полностью.
6. Вы можете запросить удаление своих данных, написав разработчику.
7. Данные используются только для модерации, аналитики и улучшения бота.
