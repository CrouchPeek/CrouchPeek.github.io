# 🤖 CrouchBotik

**CrouchBotik** — универсальный бот для Discord с экономикой, казино, дуэлями, модерацией и пасхалками. Сделает твой сервер живым и активным.
## 📜 Terms of Service (Условия использования)

### 1. Общие положения
Используя бота **CrouchBotik** (далее — «Бот»), вы соглашаетесь с этими условиями. Если вы не согласны — не используйте Бота. Всё просто.

### 2. Что можно делать
- ✅ Использовать все команды Бота в рамках их назначения.
- ✅ Приглашать Бота на свои сервера.
- ✅ Сообщать об ошибках и предлагать идеи.
- ✅ Смеяться над шутками Бота (он старается).

### 3. Что нельзя делать
- ❌ Использовать Бота для спама, оскорблений, угроз или любых действий, нарушающих правила Discord.
- ❌ Пытаться взломать Бота, его код или базу данных.
- ❌ Использовать баги в своих целях (лучше сообщить автору, он даст вам монетку).
- ❌ Выдавать себя за разработчика Бота.

### 4. Ответственность
- Бот предоставляется «как есть» (как все боты на свете).
- Разработчик не несёт ответственности за:
  - Потерю данных (хотя мы стараемся этого избегать).
  - Действия пользователей, использующих Бота.
  - Сбои Discord (это не от нас зависит).
- Бот не нарушает ваши права и не передаёт ваши данные третьим лицам (подробнее в политике конфиденциальности).

### 5. Изменения
- Мы можем обновлять эти условия в любое время.
- Актуальная версия всегда доступна в этом репозитории.
- Продолжая использовать Бота после изменений, вы автоматически принимаете новую версию.

### 6. Контакты
- По всем вопросам: Discord — `@crouchpeek1bot`
- Баги и предложения: там же, только поставьте тег `#баг` или `#идея`

### 7. Забавный пункт
- Бот имеет право на плохое настроение. Если он ответил вам сарказмом — значит, вы заслужили. 😏
- Бот любит пинг-понг. Если пингануть его слишком часто — он может ответить шуткой про резиновый сервер.



## 🔒 Privacy Policy (Политика конфиденциальности)

1. **Какие данные могут собираться:**
   - ID пользователей, серверов, каналов (только для работы бота)
   - Действия участников (баны, кики, муты, выдача/снятие ролей, изменения ника)
   - Время и частота активности
   - Команды, используемые пользователями (без сохранения контента сообщений)

2. **Какие данные НЕ собираются:**
   - Контент сообщений (текст, вложения, ссылки) — **НЕ ЛОГИРУЕТСЯ**
   - Личные сообщения (ЛС) — **НЕ ЧИТАЮТСЯ**
   - Пароли, email, телефонные номера
   - Аудио из голосовых каналов

3. **Как используются данные:**
   - Для модерации сервера
   - Для аналитики и статистики (обезличенной)
   - Для выявления нарушений
   - Для улучшения работы бота

4. **Где и как хранятся данные:**
   - Данные хранятся локально на сервере разработчика
   - Доступ к ним есть только у разработчика
   - Данные **не передаются третьим лицам**
   - Telegram используется только для уведомлений об ошибках (без передачи пользовательских данных)

5. **Управление данными:**
   - Владелец сервера может включить/отключить логирование командой `/toggle_logging`
   - Вы можете запросить удаление своих данных через Discord: `@crouchpeek1bot`
   - Команда для удаления своих данных: `/delete_my_data` (в разработке)

6. **Изменения в политике:**
   - Политика может обновляться. Актуальная версия всегда доступна в этом репозитории.

7. **Согласие:**
   - Используя бота, вы соглашаетесь с данной политикой конфиденциальности.
