## 🔒 Privacy Policy (Политика конфиденциальности)

1. **Какие данные могут собираться:**
   - ID пользователей, серверов, каналов (только для работы бота)
   - Действия участников (баны, кики, муты, выдача/снятие ролей, изменения ника)
   - Время и частота активности
   - Команды, используемые пользователями (без сохранения контента сообщений)

2. **Какие данные НЕ собираются:**
   - Контент сообщений (текст, вложения, ссылки) — **НЕ ЛОГИРУЕТСЯ**
   - Личные сообщения (ЛС) — **НЕ ЧИТАЮТСЯ**
   - Пароли, email, телефонные номера
   - Аудио из голосовых каналов

3. **Как используются данные:**
   - Для модерации сервера
   - Для аналитики и статистики (обезличенной)
   - Для выявления нарушений
   - Для улучшения работы бота

4. **Где и как хранятся данные:**
   - Данные хранятся локально на сервере разработчика
   - Доступ к ним есть только у разработчика
   - Данные **не передаются третьим лицам**
   - Telegram используется только для уведомлений об ошибках (без передачи пользовательских данных)

5. **Управление данными:**
   - Владелец сервера может включить/отключить логирование командой `/toggle_logging`
   - Вы можете запросить удаление своих данных через Discord: `@crouchpeek1bot`
   - Команда для удаления своих данных: `/delete_my_data` (в разработке)

6. **Изменения в политике:**
   - Политика может обновляться. Актуальная версия всегда доступна в этом репозитории.

7. **Согласие:**
   - Используя бота, вы соглашаетесь с данной политикой конфиденциальности.
